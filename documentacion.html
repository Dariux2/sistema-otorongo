Introducción

Versión Actual: 1.0.0
Última Actualización: Enero 2025
Estado: Producción

¿Qué es el Sistema Otorongo?

El Sistema Centro Oftalmológico El Otorongo es una solución integral diseñada para optimizar la gestión de servicios médicos oftalmológicos. Desarrollado con tecnologías web modernas, ofrece una interfaz intuitiva y funcionalidades completas para la administración de pacientes, citas, facturación y reportes.

Objetivos del Sistema

  • Automatizar la gestión de citas médicas
  • Centralizar el registro de pacientes y su historial médico
  • Optimizar el proceso de facturación
  • Generar reportes automáticos para toma de decisiones
  • Mejorar la comunicación entre el personal médico y administrativo

Características Principales

Sistema de Autenticación

Control de acceso basado en roles con tres niveles: Administrador, Médico y Recepcionista.

Gestión de Citas

Calendario interactivo con prevención automática de solapamientos y notificaciones.

Registro de Pacientes

Base de datos completa con historial médico, validaciones y búsqueda avanzada.

Facturación Automatizada

Generación de facturas con cálculo automático de IGV y múltiples métodos de pago.

Reportes en Tiempo Real

Dashboard con métricas actualizadas y gráficos interactivos para análisis.

Diseño Responsivo

Interfaz adaptable a dispositivos móviles, tablets y computadoras de escritorio.

Instalación

Requisitos del Sistema

Navegador web moderno (Chrome, Firefox, Safari, Edge)
JavaScript habilitado
Conexión a Internet (para recursos externos)
Resolución mínima: 1024x768 (recomendado: 1920x1080)

Pasos de Instalación

1

Descargar el Sistema

Descargue todos los archivos del sistema y mantenga la estructura de carpetas intacta.

sistema-otorongo/
├── index.html
├── login.html
├── dashboard.html
├── citas.html
├── pacientes.html
├── facturacion.html
├── reportes.html
├── documentacion.html
├── styles.css
├── script.js
└── assets/
2

Abrir el Sistema

Abra el archivo index.html en su navegador web preferido.

3

Acceder al Sistema

Haga clic en "Acceder al Sistema" y use las credenciales de prueba proporcionadas.

Módulos del Sistema

Dashboard

Panel de control principal que muestra estadísticas en tiempo real, citas del día, pacientes recientes y actividad del sistema.

Funcionalidades:

  • Visualización de métricas clave
  • Citas programadas para el día
  • Lista de pacientes recientes
  • Timeline de actividad
  • Notificaciones del sistema

Gestión de Pacientes

Módulo completo para el registro, búsqueda y administración de información de pacientes.

Funcionalidades:

  • Registro de nuevos pacientes con validación de datos
  • Búsqueda avanzada por nombre, DNI o teléfono
  • Visualización de historial médico
  • Edición de información del paciente
  • Vinculación con citas y facturas

Gestión de Citas

Sistema de programación de citas con calendario interactivo y prevención de conflictos.

Funcionalidades:

  • Calendario mensual interactivo
  • Programación de nuevas citas
  • Prevención automática de solapamientos
  • Filtrado por estado y médico
  • Reprogramación y cancelación de citas
  • Notificaciones de citas próximas

Sistema de Facturación

Generación y gestión de facturas con cálculo automático de impuestos.

Funcionalidades:

  • Creación de facturas con múltiples servicios
  • Cálculo automático de IGV (18%)
  • Múltiples métodos de pago
  • Impresión y exportación de facturas
  • Control de facturas pagadas y pendientes
  • Estadísticas de ingresos

Reportes y Estadísticas

Dashboard analítico con gráficos y reportes detallados del desempeño del centro.

Funcionalidades:

  • Gráficos de citas por mes
  • Análisis de ingresos mensuales
  • Distribución de servicios
  • Estado de citas
  • Pacientes frecuentes
  • Servicios más rentables
  • Resumen ejecutivo
  • Exportación de reportes

Roles de Usuario

Administrador

Permisos y Accesos:

  • Acceso completo a todos los módulos del sistema
  • Gestión de usuarios y permisos
  • Visualización de todos los reportes
  • Configuración del sistema
  • Gestión de pacientes, citas y facturación

Médico

Permisos y Accesos:

  • Visualización de sus pacientes asignados
  • Gestión de sus citas programadas
  • Acceso al historial médico de sus pacientes
  • Actualización de diagnósticos y tratamientos
  • Visualización de reportes personales

Recepcionista

Permisos y Accesos:

  • Registro de nuevos pacientes
  • Programación y gestión de citas
  • Generación de facturas
  • Gestión de pagos
  • Atención de consultas de pacientes

API y Funciones

Funciones Principales del Sistema

OtorongoSystem.getPatients()

Obtiene la lista completa de pacientes registrados.

const patients = OtorongoSystem.getPatients();
// Retorna: Array de objetos paciente

OtorongoSystem.savePatients(patients)

Guarda la lista de pacientes en LocalStorage.

OtorongoSystem.savePatients(updatedPatients);

OtorongoSystem.getAppointments()

Obtiene todas las citas programadas.

const appointments = OtorongoSystem.getAppointments();

OtorongoSystem.formatDate(dateString)

Formatea una fecha al formato local español.

const formatted = OtorongoSystem.formatDate('2025-01-15');
// Retorna: "15 de enero de 2025"

OtorongoSystem.formatCurrency(amount)

Formatea un monto al formato de moneda peruana.

const formatted = OtorongoSystem.formatCurrency(150.50);
// Retorna: "S/. 150.50"

showNotification(message, type, duration)

Muestra una notificación al usuario.

showNotification('Operación exitosa', 'success', 5000);
// Tipos: 'success', 'error', 'warning', 'info'

Preguntas Frecuentes

¿Cómo cambio mi contraseña?

Actualmente, las contraseñas deben ser cambiadas por un administrador del sistema. Contacte al soporte técnico para solicitar un cambio de contraseña.

¿Los datos se guardan automáticamente?

Sí, todos los datos se guardan automáticamente en el navegador usando LocalStorage. Sin embargo, se recomienda hacer respaldos periódicos exportando los datos.

¿Puedo acceder desde mi teléfono móvil?

Sí, el sistema tiene un diseño responsivo que se adapta a dispositivos móviles y tablets.

¿Qué navegadores son compatibles?

El sistema es compatible con las versiones más recientes de Chrome, Firefox, Safari y Edge.

¿Cómo exporto los datos del sistema?

Cada módulo tiene un botón de exportación que permite descargar los datos en formato CSV para su análisis en Excel u otras herramientas.

¿Puedo imprimir las facturas?

Sí, el sistema incluye una función de impresión optimizada para facturas. También puede guardarlas como PDF usando la función de impresión del navegador.

¿Qué hago si olvido mi contraseña?

Contacte al administrador del sistema o al soporte técnico para restablecer su contraseña.

¿El sistema tiene límite de pacientes o citas?

No hay límite técnico, pero el rendimiento puede variar según el navegador y dispositivo utilizado con grandes volúmenes de datos.
